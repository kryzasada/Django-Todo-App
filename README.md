# Django-Todo-App

<br>

Website TODO application. <br>
Written in django, HTML and CSS.


## Requirements

  * [Python 3.7.5+](https://www.python.org/downloads/)
  * [Django](https://www.djangoproject.com/)


## Run
  ```
  python manage.py runserver
  ```
